# Simulación PAS
Este proyecto implementa una simulación computacional de un Paseo Aleatorio Simple, un proceso estocástico fundamental en matemáticas y física. El objetivo principal es analizar cómo divergen múltiples trayectorias aleatorias y cómo evoluciona su comportamiento estadístico a lo largo del tiempo.

**Objetivos del Proyecto**

- Simulación masiva: Generación de múltiples trayectorias aleatorias de forma simultánea y eficiente.
- Análisis de probabilidad: Estudio de diferentes escenarios variando la probabilidad $p$.
- Estadística descriptiva: Cálculo dinámico de la media y la desviación típica de los procesos.
- Visualización de datos: Creación de gráficos avanzados que permiten comparar la evolución de cada trayectoria frente a su tendencia central.

**Tecnologías utilizadas**

Para este proyecto se han empleado las librerías :

- NumPy: Para la generación de números aleatorios y el manejo de grandes volúmenes de datos mediante vectores.
- Matplotlib: Para la representación gráfica de las trayectorias y el sombreado de los intervalos de confianza.
