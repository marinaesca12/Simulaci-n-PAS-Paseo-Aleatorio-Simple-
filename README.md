# Simulación PAS
Este proyecto implementa una simulación computacional de un Paseo Aleatorio Simple, un proceso estocástico fundamental en matemáticas y física. El objetivo principal es analizar cómo divergen múltiples trayectorias aleatorias y cómo evoluciona su comportamiento estadístico a lo largo del tiempo.

**Objetivos del Proyecto**

<u>Simulación masiva:<u> Generación de múltiples trayectorias aleatorias de forma simultánea y eficiente.
<u>Análisis de probabilidad:<u> Estudio de diferentes escenarios variando la probabilidad $p$.
<u>Estadística descriptiva:<u> Cálculo dinámico de la media y la desviación típica de los procesos.
<u>Visualización de datos:<u> Creación de gráficos avanzados que permiten comparar la evolución de cada trayectoria frente a su tendencia central.

**Tecnologías utilizadas**

Para este desarrollo se han empleado las librerías estándar en ciencia de datos :
<u>NumPy:<u> Para la generación de números aleatorios y el manejo de grandes volúmenes de datos mediante vectores.
<u>Matplotlib:<u> Para la representación gráfica de las trayectorias y el sombreado de los intervalos de confianza.
